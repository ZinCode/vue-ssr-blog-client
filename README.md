# first ssr

> A Vue.js project with vue 2.0, vue-router and vuex starter kit for server side rendering.

## Build Setup

``` bash
npm install
npm run build
npm start
```

## Devlopment Setup

```bash
npm install
npm run dev
```

## 目前仍在开发中 需要搭配后端api服务器使用
